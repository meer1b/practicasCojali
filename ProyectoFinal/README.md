# 📋 Gestor de Tareas

Aplicación web de gestión de tareas desarrollada con React y Vite como proyecto final del curso.

## ¿Qué hace?

- Crear, editar y eliminar tareas
- Asignar prioridad (alta, media, baja) y estado a cada tarea
- Filtrar tareas por estado y prioridad
- Ordenar por fecha, prioridad o título
- Los datos se guardan automáticamente en el navegador (localStorage)

## Tecnologías usadas

- **React 18** — Framework de UI
- **Vite** — Herramienta de desarrollo
- **CSS puro** — Estilos con variables CSS
- **localStorage** — Persistencia de datos

## Cómo ejecutar el proyecto

### Requisitos previos

- Tener instalado [Node.js](https://nodejs.org/) (versión 18 o superior)

### Pasos

```bash
# 1. Instalar dependencias
npm install

# 2. Arrancar el servidor de desarrollo
npm run dev
```

Luego abre el navegador en: **http://localhost:5173**
